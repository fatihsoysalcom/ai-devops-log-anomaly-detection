# AI DevOps Log Anomaly Detection

This Python script simulates real-time system metrics and applies a basic anomaly detection algorithm. It demonstrates how AI can enhance DevOps monitoring by automatically identifying unusual patterns in operational data, such as error rates or response times. This simplified example highlights the concept of AI-driven proactive issue detection, a core theme of AI-DevOps integration.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`
